# Target Evaluation Bioinformatics Training

Beginner-friendly training project for learning TCGA/GTEx/DepMap data handling, harmonization, and reproducible bioinformatics workflows.

## Week 1 Goal
- Set up project structure
- Learn TCGA and GTEx dataset structure
- Load and inspect expression + metadata in Python/R
- Perform basic cleaning and harmonization prep

## Project Structure
- data/raw          Raw downloaded files (do not edit)
- data/processed    Cleaned/derived files
- metadata          Sample and dataset metadata
- scripts           One script per step
- notebooks         Exploration notebooks
- results           Output tables
- figures           Plots
- docs              Notes and summaries
- env               Environment files
